## Description

The `ft_printf` project is a reimplementation of the standard C `printf` function.  
Its goal is to support formatted output with multiple conversions including characters, strings, integers, unsigned integers, pointers, and hexadecimal numbers.

---

## Instructions

Clone the repository and compile the project:

```bash
git clone <repository_url>
cd ft_printf
make
```
